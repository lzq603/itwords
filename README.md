# itwords
IT术语词典  
一款服务于非专业人士的科普小程序，程序自动地从互联网上爬取信息领域热词及其释义和引用到该词汇的相关文章等存入数据库，供用户查询。  

### 项目说明  

- 词云：以词云形式显示目前IT界流行词汇  
- 新词：以卡片形式展示所爬取到的最新词汇，点击卡片获取详细信息  
- 搜索：执行此查找  

程序使用JAVA语言、Springboot框架做程序后台，结合Python语言爬取信息，使用微信小程序语言(wxml、wxss、js)做微信小程序客户端（前台）
